段落与链接
web-booster 用原生 Custom Elements 提供跨技术栈可复用 UI。你可以在一段正文里直接穿插 Web Components 文档 或者把它接进 Vue、React、ASP.NET 页面。 对外暴露的设计令牌统一使用 --wb-* 前缀，组件标签统一使用 wb-* 前缀。
